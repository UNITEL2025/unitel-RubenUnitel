Crear una aplicación de FAQ (Frequently Asked Questions), Preguntas Frecuentes.
Mostrará un HTML con un acordeón, se entregará el HTML base.

La aplicación debe realizar:
1) Crear una base de datos
2) Rellenar con datos la base de datos (título, texto del FAQ y contador con número de veces que ha aparecido en las búsquedas)
3) Mostrar una pantalla con un input text, donde el user pueda escribir su duda, el sistema deberá buscar en la base de datos, y cargar el acordeón correspondiente, si existen resultados
4) En caso de error el sistema debe mostrar un alert danger en pantalla (añadir un botón que lance el error)
5) Si no existen resultados, deberá mostrar un alert info avisando al user
Todas las operaciones se deben realizar sin recargar la página web, haciendo uso de AJAX
